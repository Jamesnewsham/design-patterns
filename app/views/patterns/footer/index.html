HMRC Footer

These patterns are work in progress.

View the todo list for this pattern on GitHub.

The Footer used on all HMRC services should be the standard GOV.UK footer, with the addition of HMRC spesific links.

Footer design

The standard footer should be used on all services.

The report a problem link 'Is there anything wrong with this page?' is intended to be used by users to tell HMRC if there is something technically wrong with the page. However the design of this component is misleading and attracts contact from users who have a problem using the service.

The following utility links should be included:

  • cookies
  • privacy policy
  • terms and conditions
  • help using GOV.UK

Cookies

Point to HMRC's cookie page https://www.tax.service.gov.uk/help/cookies.

Privacy policy

Point to HMRC's privacy policy https://www.tax.service.gov.uk/help/privacy.

Terms and conditions

Point to HMRC's terms and conditions https://www.tax.service.gov.uk/help/terms-and-conditions.

Help using GOV.UK

Point to HMRC's help using GOV.UK https://www.gov.uk/help.
