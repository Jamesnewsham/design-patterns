HMRC Header

The HMRC header is an variant of the GOV.UK header.

Header design

The HMRC logo is included for services where there is a user need to reassure users that they are using an HMRC service. For example, when they are making a payment and need to be sure that the money is going to the right governmant department.

More user research is needed to prove if the HMRC logo reassures users.

  • Header - signed out

Header signed out

Should be used for services when there is a need to identify the service as an HMRC service.

This version of the header should contain:

  • the GOV.UK header
  • a phase banner unless the service is live
  • HMRC logo
  • Welsh toggle

Header signed in

Should be used for services when there is a need to identify the service as an HMRC service and when users can have a signed in state.

This version of the header should contain:

  • the GOV.UK header
  • a sigin out link
  • a phase banner unless the service is live
  • HMRC logo
  • Last logged in status
  • Welsh toggle
